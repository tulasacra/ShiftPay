About ShiftPay

ShiftPay creates a fixed-rate SideShift shift via the API and then gives you a BCH wallet deep link for the exact deposit. Each user brings their own SideShift API keys (stored only in this browser).

SideShift API keys

A SideShift account is created when you visit the account page. Copy your private key and account ID (affiliate id) here. They never leave your device except when calling SideShift.
